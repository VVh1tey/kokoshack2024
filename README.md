## Описание репозитория


## Структура репозитория
``` 
├── data
│   ├── raw/                 # Сырые данные, которые не обрабатываются напрямую
│   └── processed/           # Предобработанные данные для моделей
│
├── notebooks/               # Jupyter Notebooks
│   ├── 00_eda_artem.ipynb   # Первичный анализ данных
│   ├── 01_eda.ipynb         # Первичный анализ данных
│   ├── 02_feature_engineering.ipynb  # Работа с фичами
|   ....
│   └── 0n_modeling_experiments.ipynb # Моделирование и эксперименты
│
├── src/                     # Python-скрипты для пайплайнов
│   ├── preprocess.py        # Предобработка данных
│   ├── train_model.py       # Обучение модели
│   └── evaluate_model.py    # Оценка модели и генерация выходных данных
│
├── output/                  # Результаты моделей
│   ├── predictions/         # Предсказания моделей
│   └── logs/                # Логи моделей
│
├── models/                  # Сохраненные модели
│   └── model_name.pkl
│
├── experiments/             # Эксперименты, зафиксированные в MLFlow или другими способами
│   └── mlflow/              # Директория для артефактов MLFlow
│
├── requirements.txt         # Список зависимостей
├── dvc.yaml                 # DVC пайплайн для обработки данных и обучения моделей
├── README.md                
└── .gitignore               
```
